# Fox Jumper - 2D Game Built w/ Unity and C#

**Game can be played here:** https://jamesfoti.github.io/FoxJumper2dVideoGame/


## Game Controls
**Run:** "Left Arrow" or "s" (Run Left) and "Right" or "d" (Run Right) <br/>
**Jump:** "Space" <br/>
**Pause:** "p" or "ESC" <br/>
